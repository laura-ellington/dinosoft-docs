    Quickstart

    Get DinoSoft installed and run your first analysis in under five minutes.

    Prerequisites: You need Python 3.10 or later. Check your version with python --version.

    1. Install DinoSoft

    pip install -e .

    If you prefer to use a virtual environment (recommended):

    python -m venv .venv
source .venv/bin/activate  # On Windows, use .venv\Scripts\activate
pip install -e .

    2. Run the demo

    python -m dinosoft

    You should see output like this:

    ==================================================
  DinoSoft  -  Dinosaur Dietary Analysis
==================================================

Diet breakdown:
  carnivore     3
  herbivore     3
  omnivore      1

Top 3 biggest eaters (kg/day):
  Brachiosaurus        400 kg/day  (HF=400.0)
  Tyrannosaurus        230 kg/day  (HF=230.0)
  Triceratops          200 kg/day  (HF=200.0)

    3. Try it in Python

    from dinosoft import load_sample_data, diet_summary

data = load_sample_data()  # Loads the built-in dataset of 7 dinosaur species
print(diet_summary(data))  # Returns {'herbivore': 3, 'carnivore': 3, 'omnivore': 1}

    Done!

    You now have DinoSoft installed with the sample dataset loaded. Head to the Tutorial to learn what you can do with it.

    Something not working? Make sure you are in the project root directory (where pyproject.toml is) before running pip install -e .
